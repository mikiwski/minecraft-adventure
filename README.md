# Minecraft Adventure Tasks Mod

Mod dla Minecraft 1.21.1 dodający system zadań z rosnącą trudnością.

## Funkcje

- 200 predefiniowanych zadań z rosnącą trudnością
- System nagród (XP + przedmioty) proporcjonalny do trudności
- HUD overlay wyświetlający 5 zadań (2 poprzednie, 1 aktywne, 2 następne)
- GUI ekran z pełną listą zadań, historią i statystykami
- Wsparcie dla wariantów przedmiotów (np. dowolny rodzaj drewna)
- Wielojęzyczność: polski i angielski

## Wymagania

- Minecraft 1.21.1
- Fabric Loader 0.16.0+
- Fabric API

## Instalacja

1. Zainstaluj Fabric Loader dla Minecraft 1.21.1
2. Skopiuj plik `.jar` moda do folderu `mods`
3. Uruchom grę

## Użycie

- Naciśnij klawisz `T` (domyślnie) aby otworzyć GUI zadań
- Zadania są automatycznie śledzone podczas gry
- Nagrody są przyznawane automatycznie po ukończeniu zadania

## Rozwój

```bash
./gradlew build
```
